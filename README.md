# Placement Status Prediction Using Logistic Regression

A beginner machine learning project that predicts whether a student is likely to be **Placed** or **Not Placed** using Logistic Regression.

---

## Project Overview

This project creates a synthetic student dataset and trains a Logistic Regression model to predict placement status based on student performance details.

This project helps in learning:

- Dataset creation
- Data preprocessing
- Train-test split
- Label Encoding
- Logistic Regression
- Model evaluation
- Prediction
- Probability prediction

---

## Features Used

| Feature              | Description |
| Attendance           | Student attendance percentage |
| Coding Skills        | Coding skill score from 0 to 10 |
| Study Hours          | Study hours score from 0 to 10 |
| Communication Skills | Communication skill score from 0 to 10 |

---

## Target Variable

| Value        | Meaning |
| `placed`     | Student is predicted as placed |
| `not placed` | Student is predicted as not placed |

---

## Dataset

This project uses a **synthetic dataset** created using Python's `random` library.

- Number of student records: 300
- Dataset type: Classification dataset
- Target column: `placement_status`

The placement status is generated using attendance, coding skills, study hours, and communication skills.

---

## Algorithm Used

- Logistic Regression

Logistic Regression is used because the output has two classes:

text
`placed`
`not placed`


## Important Note

This project uses a synthetic dataset created only for educational purposes.

The model should not be used for real placement decisions because actual placement depends on many factors such as interviews, company requirements, projects, aptitude tests, academic performance, and communication skills.
